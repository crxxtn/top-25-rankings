# Top 25 College Football Rankings European Friendly Website

Currently live at https://ncaatop25cs.netlify.app

Made by Christian Stephens with Tailwind and Svelte Kit

### Setup

`npm i`

### Dev

`npm run dev` or `npm run dev --open`

### Build and Preview

`npm run build`

`npm run preview`
